-------How to use a client ?--------
1.Type information about a user:[Student/Profesor] [Nume] [Prenume] ATM [age] [an/materie]
2.Type 'getHumans' to get all the users from server and 2 ENTERs
3.Type 'exit' to close socket connection with server
